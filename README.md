Мобільна аналітична система розподілу і керування задачами
Основні можливості:
- Опис ключових функцій (наприклад, створення, редагування, видалення задач)
- Інтеграція з Google Calendar для нагадувань
- Авторизація користувачів через Google Sign-In
- Підтримка статусів задач (нові, в процесі, виконані, прострочені)
- Можливість додавання файлів/вкладень до задач

Використані технології:
- Java для Android
- Android Studio
- Google APIs (Calendar, Sign-In)
- Volley для роботи з REST API
- MySQL
- Node.js

Інструкції з встановлення:
- Клонуйте репозиторій: git clone https://github.com/yourusername/yourproject.git
- Відкрийте проект в Android Studio
- Налаштуйте необхідні API ключі та OAuth-креденціали (Google Cloud Console)
- Запустіть додаток на емуляторі або реальному пристрої
